Análise e Projeto de Sistemas

  • Início

Em Análise e Projeto de Sistemas no primeiro ano, você tem uma introdução aos conceitos fundamentais de informática e tecnologia da informação, além de noções básicas sobre o desenvolvimento de sistemas.

Alguns tópicos que podem ser abordados são:

Lógica de Programação: Introdução a lógica de programação, que envolve conceitos como algoritmo, fluxograma, variáveis, etc.

Fundamentos de Sistemas de Informação: Conceitos básicos sobre o que são sistemas de informação, como eles são utilizados nas empresas e a importância de uma boa análise para o desenvolvimento do sistema.

Modelagem de Dados: Introdução a modelagem de dados, com foco em como organizar e estruturar informações usando diagramas.

Análise de Requisitos: Primeiros conceitos de levantamento de requisitos, onde se aprende a identificar as necessidades dos usuários.

Metodologias de Desenvolvimento de Sistemas: Introdução a algumas metodologias ágeis e tradicionais (como SCRUM), mostrando como os projetos de software podem ser organizados e desenvolvidos.

Esses tópicos servem como base para os anos seguintes, nos quais você vai se aprofundar mais no desenvolvimento de softwares, análise de sistemas e banco de dados.
