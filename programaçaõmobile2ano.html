Programação Mobile

  • Início

Em Programação Mobile no segundo ano do ensino médio técnico, geralmente você aprende os seguintes conceitos e habilidades:

Fundamentos da Programação Mobile: inclui introdução às plataformas de desenvolvimento de aplicativos móveis, como Android e iOS e Linguagens de Programação como Java/Kotlin e JavaScript.

Interface de Usuário (UI) e Experiência do Usuário (UX): que fala sobre Design Responsivo como a criação de interfaces que se adaptam a diferentes tamanhos de tela e sobre Componentes Visuais como Botões, listas, formulários, e layouts básicos.

Desenvolvimento de Aplicações Básicas: aprendemos sobre a estrutura de um App Mobile, como estruturar e organizar o código de um aplicativo e a Manipulação de Dados Simples, como Manipular dados utilizando banco de dados locais, como SQLite ou Shared Preferences.

Design de Interfaces: Princípios de design responsivo e boas práticas para criação de interfaces de usuário (UI) em dispositivos móveis.

Esses tópicos te preparam para criar seus primeiros aplicativos móveis funcionais e conhecer as boas práticas do mercado.
